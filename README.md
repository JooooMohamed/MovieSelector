## Getting Started

For you run this project you will need to install it locally using npm.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/JooooMohamed/MovieSelector.git 
   
2. Install NPM packages
   ```bash
   npm install  
   ```

3. Run start command on npm
   ```bash
   npm start
   ```
